# Mao-Chronicles-1949-1976

> Mao Zedong Chronicles 1949-1976

## Project Overview

This repository systematically documents major events and activities of Mao Zedong from 1949 to 1976, organized by year.

## Directory Structure

- `chronicles/` - Year-by-year chronicles

## Period Classification

| Period | Years |
|--------|-------|
| Early PRC | 1949-1952 |
| Socialist Transformation | 1953-1956 |
| Building Socialism | 1957-1965 |
| Cultural Revolution | 1966-1976 |
